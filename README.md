# gestion-incidentes-mitre
Análisis de la respuesta a un incidente simulado, utilizando el marco de clasificación MITRE ATT&amp;CK

## Proyecto: Clasificación y Respuesta a Incidentes con MITRE ATT&CK

Este repositorio se enfoca en la fase de **Análisis de Impacto y Respuesta** dentro del ciclo de vida de un incidente.

## Habilidades Demostradas:
* **Gestión de Incidentes (IR):** Aplicación de un plan estructurado de respuesta.
* **Cumplimiento y Procesos:** Uso del marco **MITRE ATT&CK** para clasificar la táctica y la técnica del atacante.
* **Habilidades Blandas:** Comunicación clara y documentación técnica.

## Contenido:
1.  `caso_phishing.pdf` o `caso_phishing.md`: Documento que describe un caso de phishing simulado.
2.  `analisis_mitre.md`: Clasificación de cómo el atacante ganó acceso (Táctica) y qué hizo (Técnica) según MITRE.

## Proceso:
El análisis demuestra cómo un analista clasificaría la amenaza para generar un reporte que facilite la mitigación y las acciones preventivas futuras.
